建议使用"pnpm + Electron + React + TypeScript + Claude Code SDK"（具体请根据项目自行决定）
提示：由于这是一个完全的私人项目，所以允许你直接使用参考项目中的代码，允许直接复制

   - 集成oh-my-opencode的多LLM智能体协同编程框架，实现多AI模型、多代理的协同，并行后台任务工作机制，Hook 生命周期治理，持续执行/自动续跑

   - 支持自定义api模型接入，需兼容OpenAI-compatible
3. 界面设计：
     * 采用hello-halo的UI界面设计风格，实现多视图并行的工作台布局
