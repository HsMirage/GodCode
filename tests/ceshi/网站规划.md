### 二、详细功能清单

#### 🏠 1. 首页
- 网站Logo和品牌标语
- 数据统计展示（收录品牌数、黑榜事件数、红榜事件数）
- 今日热门事件滚动展示
- 最新收录事件（黑榜/红榜分栏）
- 品牌行业分类导航
- 快速搜索入口

#### ⚫ 2. 黑榜页面
- 负面事件列表（卡片式/列表式切换）
- 排序：时间、热度、严重程度
- 筛选：行业、国家、时间范围
- 标签云展示
- 无限滚动/分页

#### 🔴 3. 红榜页面
- 正面事件列表
- 排序：时间、热度、贡献程度
- 筛选：行业、国家、时间范围
- 品牌荣誉榜

#### 🔍 4. 搜索功能
```
搜索维度：
├── 关键词搜索（品牌名、事件内容）
├── 时间范围筛选（开始日期 - 结束日期）
├── 品牌筛选（下拉选择/模糊搜索）
├── 事件类型（黑榜/红榜）
├── 行业分类（科技、食品、服装、汽车等）
├── 涉及国家/地区
└── 事件标签（辱华、抄袭、慈善、环保等）
```

### 三、数据库设计

```sql
-- 数据库表结构设计

-- 1. 用户表
users
├── id (主键)
├── username (用户名)
├── email (邮箱)
├── password_hash (密码哈希)
├── avatar (头像)
├── role (角色: user/admin/super_admin)
├── status (状态: active/banned)
├── created_at
└── updated_at

-- 2. 品牌表
brands
├── id (主键)
├── name (品牌名)
├── english_name (英文名)
├── logo (品牌Logo)
├── country (所属国家)
├── industry (行业分类)
├── description (品牌简介)
├── black_count (黑榜事件数)
├── red_count (红榜事件数)
├── created_at
└── updated_at

-- 3. 事件表
events
├── id (主键)
├── brand_id (关联品牌)
├── user_id (发布用户)
├── type (类型: black/red)
├── title (事件标题)
├── content (事件详情)
├── event_date (事件发生日期)
├── event_location (事件地点)
├── affected_country (涉及国家)
├── severity (严重程度1-5)
├── source_urls (来源链接JSON)
├── status (状态: pending/approved/rejected)
├── reject_reason (拒绝原因)
├── view_count (浏览量)
├── created_at
└── updated_at

-- 4. 事件图片表
event_images
├── id (主键)
├── event_id (关联事件)
├── image_url (图片地址)
├── sort_order (排序)
└── created_at

-- 5. 标签表
tags
├── id (主键)
├── name (标签名)
├── type (类型: black/red/both)
└── usage_count (使用次数)

-- 6. 事件标签关联表
event_tags
├── event_id
└── tag_id

-- 7. 评论表
comments
├── id (主键)
├── event_id (关联事件)
├── user_id (评论用户)
├── parent_id (父评论ID，支持回复)
├── content (评论内容)
├── status (状态)
├── created_at
└── updated_at

-- 8. 收藏表
favorites
├── id (主键)
├── user_id
├── event_id
└── created_at

-- 9. 审核日志表
audit_logs
├── id (主键)
├── event_id (关联事件)
├── admin_id (审核管理员)
├── action (操作: approve/reject)
├── reason (原因)
└── created_at

-- 10. 行业分类表
industries
├── id (主键)
├── name (行业名)
├── icon (图标)
└── sort_order (排序)
```
